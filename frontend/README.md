# Frontend

Static UI for dormitory management.

## Main files

- `index.html` and page HTML files
- `style.css`
- `tailwind-config.js`
- `js/`
- `components/`
- `data/` (CSV seed datasets)

## Run only frontend

From project root:

```powershell
python -m http.server 4173
```

Open `http://127.0.0.1:4173/frontend/index.html`.
